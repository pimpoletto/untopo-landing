# Politique de confidentialité

Dernière mise à jour : 2026-06-12

## 1. Responsable du traitement

Le service Untopo est édité par une entité dont les coordonnées complètes seront publiées dans les mentions légales.

Contact : contact@untopo.com

## 2. Données traitées

Untopo traite notamment :

- données de compte (email, identifiants, profil) ;
- données de rapport (notes vocales transcrites, photos, annotations, exports PDF) ;
- données de personnalisation (nom d'entreprise, logo, signature) ;
- données techniques (logs, diagnostics) ;
- données de facturation (statut d'abonnement via Stripe).

## 3. Finalités

Les données sont traitées pour fournir l'application, créer et exporter les rapports, gérer l'abonnement et sécuriser le service.

## 4. Sous-traitants

Les données peuvent être traitées par Supabase, Vercel, OpenAI, Resend, Stripe et Sentry, dans la mesure nécessaire à leurs services respectifs.

## 5. Droits des personnes

Conformément au RGPD, vous pouvez demander l'accès, la rectification, l'effacement ou la limitation de vos données en écrivant à contact@untopo.com.
